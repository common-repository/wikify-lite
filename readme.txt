Allow users to create and update posts using a front-end editor.

== Description ==

Turn WordPress into a wiki site (that blends into your existing WordPress theme).

== Installation ==

1. Upload and install the plugin via zip file.
2. Adjust settings (optional) wp-admin->settings->wikify
3. Visit http://yoursite.com/wiki/ to view archive (or user the slug you entered on the settings page).


== Features ==
* Front-end WYSIWYG editor
* Front-end post history table
* Restore post revisions
* Ajax Comments

== Pro Features ==
* Wiki post contributors list (widget)
* Media uploads
* Advanced Custom Fields support
* User Permissions
* Pending Posts (published post remains visible while changes are hidden, awaiting review)
* Multiple custom post types support
* Wiki tag and category taxonomies
* Customizable default wiki parameters
